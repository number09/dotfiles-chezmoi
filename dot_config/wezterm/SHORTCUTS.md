# WezTerm ショートカットチートシート

## タブ操作

| 操作 | ショートカット |
|------|----------------|
| 新しいタブを作成 | `CMD + T` |
| タブを閉じる | `CMD + W` (パネルが1つの場合) |
| 次のタブに移動 | `CMD + Shift + ]` または `CTRL + TAB` |
| 前のタブに移動 | `CMD + Shift + [` または `CTRL + Shift + TAB` |
| タブ番号で移動 (1-9) | `CMD + 1~9` |

## その他の便利な操作

| 操作 | ショートカット |
|------|----------------|
| フォントサイズを大きく | `CMD + +` |
| フォントサイズを小さく | `CMD + -` |
| フォントサイズをリセット | `CMD + 0` |
| コピー | `CMD + C` |
| ペースト | `CMD + V` |
| 検索 | `CMD + F` |
| ウィンドウを閉じる | `CMD + Q` |

## 現在の設定情報

- **フォント**: Ricty Diminished (14pt)
- **カラースキーム**: iceberg-dark
- **初期サイズ**: 120列 × 28行
- **自動リロード**: 無効

---

## カスタマイズ

ショートカットを変更したい場合は、[wezterm.lua](wezterm.lua) の `config.keys` セクションを編集してください。

設定の詳細については [WezTerm 公式ドキュメント](https://wezfurlong.org/wezterm/config/files.html) を参照してください。
